# Regulatory Repository UI/UX Improvements Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Improve the generated static Regulatory Repository UI — reader readability, result-card scanning, mobile filter behavior, and keyboard accessibility — while preserving the restrained Honda-style industrial design.

**Architecture:** Static app. `scripts/build.py` (Python/Jinja2/bleach) generates `dist/data/*.json` and renders the shell from `templates/index.html.j2`; `assets/app.js` (vanilla JS + MiniSearch) and `assets/styles.css` are copied to `dist/assets/`. Source-of-truth edits happen in `assets/`, `templates/`, `scripts/`; `dist/` is regenerated by `python scripts/build.py` and never hand-edited. Only `scripts/build.py` is unit-testable via `pytest`; JS/CSS changes are verified by rebuilding and rendered smoke checks against `dist/index.html`.

**Tech Stack:** Python 3.11, Jinja2, bleach, pytest, ruff; vanilla JS, CSS container queries, MiniSearch; local browser smoke testing.

**Provenance:** Derived from `cross-agent-review/regulatory-repository-ui-ux-implementation-plan.md`, which itself corrected Codex's `regulatory-repository-ui-ux-final-plan.md`. The five DELTAs from that assessment are folded in and labelled inline.

- [ ] **Step 3: Implement the cleanup helper (DELTA-1 narrowed regex)**

In `scripts/build.py`, add immediately above `def summarize` (line 277):

```python
# Only the workbook stub scaffolding labels. Do NOT include generic
# "Source"/"Notes" — they occur in real legal bodies and summarize() runs on
# every record, not just stubs (would truncate genuine summaries mid-sentence).
SUMMARY_SCAFFOLD_RE = re.compile(
    r"\s+(?:Regulated Area|Applicability):\s+",
    re.IGNORECASE,
)


def clean_summary_display_text(plain: str) -> str:
    text = unescape(re.sub(r"\s+", " ", plain or "")).strip()
    match = SUMMARY_SCAFFOLD_RE.search(text)
    if match and match.start() >= 40:
        return text[: match.start()].rstrip(" .;:-")
    return text
```

Then update `summarize` so cleanup runs before truncation:

```python
def summarize(body_html: str) -> str:
    plain = bleach.clean(body_html, tags=[], strip=True)
    plain = unescape(re.sub(r"\s+", " ", plain)).strip()
    plain = clean_summary_display_text(plain)
    if len(plain) <= 250:
        return plain
    cutoff = plain.rfind(" ", 0, 250)
    if cutoff < 180:
        cutoff = 250
    return plain[:cutoff].rstrip() + "..."
```

```js
    function baseSearchText(record) {
      return [
        record.title, record.citation,
        (record.aliases || []).join(" "),
        record.summary_text,
        (record.un_equivalent || []).join(" "),
        (record.tags || []).join(" "),
        (record.open_tags || []).join(" "),
      ].join(" ");
    }

    function bodyMatchSnippet(record, query) {
      const q = normalize(query).trim();
      if (!q || q.length < 3 || normalize(baseSearchText(record)).includes(q)) return "";
      const text = searchDocsById.get(record.id) || "";
      const lower = normalize(text);
      const idx = lower.indexOf(q);
      if (idx < 0) return "";
      const start = Math.max(0, idx - 90);
      const end = Math.min(text.length, idx + q.length + 140);
      const prefix = start > 0 ? "..." : "";
      const suffix = end < text.length ? "..." : "";
      return `${prefix}${text.slice(start, end).trim()}${suffix}`;
    }

    function cardSummaryHtml(record, query) {
      const snippet = bodyMatchSnippet(record, query);
      if (snippet) return `<p class="summary summary-snippet">${highlight(snippet, query)}</p>`;
      return `<p class="summary">${highlight(record.summary_text || "No summary available.", query)}</p>`;
    }
```

```js
    (function () {
      const tip = document.getElementById("tip");
      function showTip(el) {
        tip.textContent = el.dataset.tooltip;
        tip.style.display = "block";
        tip.setAttribute("aria-hidden", "false");
        const r = el.getBoundingClientRect();
        const tw = tip.offsetWidth;
        const left = (r.right + 10 + tw > window.innerWidth) ? r.left - tw - 10 : r.right + 10;
        tip.style.left = left + "px";
        tip.style.top  = Math.max(8, r.top - 2) + "px";
      }
      function hideTip() { tip.style.display = "none"; tip.setAttribute("aria-hidden", "true"); }
      document.addEventListener("mouseover", (e) => { const el = e.target.closest("[data-tooltip]"); if (el) showTip(el); else hideTip(); });
      document.addEventListener("mouseout",  (e) => { if (!e.relatedTarget || !e.relatedTarget.closest("[data-tooltip]")) hideTip(); });
      document.addEventListener("focusin",   (e) => { const el = e.target.closest("[data-tooltip]"); if (el) showTip(el); });
      document.addEventListener("focusout",  (e) => { if (!e.relatedTarget || !e.relatedTarget.closest("[data-tooltip]")) hideTip(); });
    })();
```
